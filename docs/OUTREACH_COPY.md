# Outreach copy for community playtesting

Replace bracketed placeholders before posting:

- `[PLAY_LINK]` - public itch/game URL.
- `[REVIEWER_SIGNUP_LINK]` - form, Discord thread, email, or issue for reviewer volunteers.
- `[SCREENSHOT_OR_GIF]` - optional image/video attachment.

## Core positioning

Use this framing consistently:

> Help shape a weird little arcade game. Play one short run, submit feedback from the end screen, and community volunteers will help triage what should change next.


## Posting notes

- Lead with the 5-minute ask.
- Attach a GIF or screenshot when possible.
- Do not overexplain the backend or reviewer workflow in player posts.
- Post the reviewer ask separately when recruiting volunteers.
- Reply to early comments quickly; the first hour matters.
- After the first batch, post a recap so people see feedback became action.
