# First session — honesty brief

**Audience:** new players and playtesters  
**Purpose:** set expectations for the first short run (tutorial honesty / no dark patterns)

## What this game is

- A short **roguelike idle arcade** toy: four guys, sticky PB blobs, tap to chomp.
- Runs are quick. Meta (Crust Credits) may be stubbed depending on build.

## What you will **not** see

- FOMO timers that punish you for stepping away mid-run framing as failure-shame
- Pay-to-progress or loot-box monetization
- Forced account to keep playing the local build

Five misses → **Stuck Shut** is a run-end condition, not a streak-shame system. Feedback send is **optional**.

## Suggested first 3–5 minutes

1. Pick a **Lunch modifier** on the start overlay.
2. Tap **Carl / Dave / Ben / Ed** to chomp nearest blobs; try for a short Frenzy chain.
3. Finish a run (win or Stuck Shut), then optionally **Send feedback** if you want.
4. Read `docs/PLAYTEST_FEEDBACK.md` only if you are helping with community review.
